Testing Overview

The Affiliate Matrix project includes comprehensive test cases for both backend and frontend components. These tests ensure that all components work correctly individually and together as a system.

The testing strategy follows a pyramid approach:

  • Unit Tests: Test individual functions and components in isolation
  • Integration Tests: Test interactions between components
  • End-to-End Tests: Test complete user workflows

Backend Tests

Backend tests are implemented using pytest and the FastAPI TestClient. They cover all API endpoints and ensure that the backend behaves correctly under various conditions.

Programs API Tests

Tests for the Programs API endpoints:

  • test_list_programs: Test retrieving a paginated list of programs
  • test_list_programs_with_filters: Test retrieving programs with filters
